# java_n8n

Java Swing contact form that sends user input (name/email/query) to an n8n webhook.

## OOP Structure
- **`User`**: Model for user form data.
- **`WebhookService`**: Sends user data to the n8n webhook.
- **`AppController`**: Handles the data submission and business logic.
- **`AppFrame`**: Swing GUI for data entry and submit action.
